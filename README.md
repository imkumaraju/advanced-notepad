# Advanced Notepad - Chrome Extension

A premium, modern note-taking extension built with React, TypeScript, and Tiptap.

## Features
- **Rich Text Editing**: Full formatting support (Bold, Italic, Underline, Strike, Alignment, Lists).
- **Auto-save & Persistence**: Data is saved locally using `chrome.storage.local`.
- **Note Management**: Create, view, and delete notes. Notes are automatically renamed based on their content.
- **Social Login**: Sign in with Google, GitHub, or Outlook (powered by Firebase Auth).
- **Cloud Backup**: Architecture for backing up data to AWS S3.

## Tech Stack
- **Frontend**: Vite + React + TypeScript
- **Editor**: Tiptap (Headless ProseMirror wrapper)
- **Icons**: Lucide React
- **Auth**: Firebase Authentication
- **Storage**: Chrome Storage API / AWS S3

## Setup Instructions

### 1. Installation
```bash
npm install
```

### 2. Configuration
Open `src/services/auth.ts` and replace the `firebaseConfig` object with your actual credentials from the Firebase Console.
Make sure to enable **Google**, **GitHub**, and **Microsoft** auth providers in Firebase.

### 3. Development
```bash
npm run dev
```

### 4. Build for Production
```bash
npm run build
```

### 5. Load in Chrome
1. Go to `chrome://extensions`
2. Enable **Developer mode**
3. Click **Load unpacked**
4. Select the `dist` folder generated by the build.

## Project Structure
- `src/components`: UI components (Sidebar, Editor, Modals, etc.)
- `src/services`: Abstractions for Storage, Auth, and Backup.
- `src/types`: TypeScript interfaces.
- `public/icons`: Generated app logos.
